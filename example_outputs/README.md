# Example Outputs

This directory contains example output files generated by LeakPy during tests on leakix.net.

These files serve as examples to show the format of data returned by the LeakIX API.

## Files

- `bulk_results.txt` - Example of bulk search results
- `domain_details.json` - Example of domain details
- `host_details.json` - Example of host details
- `subdomains.json` - Example of subdomain list
